String formatting (3 styles)¶

Percent-style (%) string formatting (old style)

print("The answer is %6.4f +/- %8.3f" % (y, dy) )

str.format() (new style)

print("The answer is {:6.4f} +/- {:8.3f}".format(y, dy))

f-strings

print(f"The answer is {y:6.4f} +/- {dy:8.3f}")

The n in {dy:n.3f} is the minimum field width

  • Specifies minimum number of characters to display a value.
  • If the actual value has fewer the field will be padded with spaces to meet minimum width.
  • If the actual value has more characters, minimum field width is ignored and the full value is displayed.
  • n does not have to be specified, e.g., {dy:.3f}
